# ⚡ **Auto Tap System Documentation**\n\n## 📋 **OVERVIEW**\n\nThe Auto Tap system allows players to purchase automated STG (ton) generation plans that run for specified durations and automatically generate rewards. This creates a passive income system that encourages long-term engagement and provides strategic resource management.\n\n---\n\n## 🎯 **KEY FEATURES**\n\n### **📦 Plan Tiers**\n- **Basic Plans** ⚡ - Affordable entry-level plans\n- **Premium Plans** 💎 - Enhanced rewards for dedicated players\n- **Elite Plans** 👑 - High-value plans for serious players\n- **VIP Plans** 🌟 - Maximum rewards for elite players\n\n### **⏰ Duration Options**\n- **Short-term**: 6 hours, 12 hours\n- **Medium-term**: 24 hours, 48 hours\n- **Long-term**: 72 hours, 7 days, 30 days\n\n### **💰 Reward Structure**\n- **Base Rewards**: 2-2500 STG per plan\n- **Hourly Rates**: 0.33-3.47 STG per hour\n- **Bonus Multipliers**: Up to 3x for VIP plans\n- **Faction Bonuses**: Iran (10%) vs USA (5%)\n\n---\n\n## 📊 **PLAN
